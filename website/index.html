Adaptive Layout

Адаптивна верстка

Automatically switches between Wide, Medium, and Small layouts depending on terminal width, avoiding messy line wrapping.

Автоматично перемикається між трьома режимами (Wide, Medium, Small) відповідно до ширини вашого вікна, запобігаючи негарному розриву рядків.

Smart Git Tracking

Розумний Git-трекінг

Directly queries local Git binary to retrieve active branch and dirty/clean status with zero delay.

Працює безпосередньо з локальним клієнтом Git, повертаючи поточну гілку та стан dirty/clean миттєво і точно.

Dynamic Quota Swap

Динамічні ліміти LLM

Detects the active AI model and automatically shows 5-hour and weekly quota usage along with reset countdowns.

Визначає активну модель штучного інтелекту та перемикає лічильники 5-годинної та тижневої квоти разом із часом до їх скидання.

Sandbox Indicators

Індикатори Сендбоксу

Visually reports sandbox state and network access settings: ON (net), ON (no-net), or fully disabled.

Візуально відображає мережевий статус вашого безпечного оточення: ON (net), ON (no-net) або повністю вимкнено.

Контекст та Токени

Має графічний 20-сегментний індикатор заповнення контекстного вікна разом із лічильником вхідних/вихідних токенів у форматі K/M.

Simple Installation

Просте встановлення

Requires no root/sudo privileges. Installs cleanly within your user profile directory.

Не вимагає root доступу чи глобального втручання. Встановлюється виключно в межах локального профілю користувача.

1

Install via curl/wget

Встановлення через curl/wget

Copy and execute the automated setup script in your Bash/Zsh terminal:

Скопіюйте та виконайте автоматичний скрипт встановлення у вашому Bash/Zsh терміналі:

curl -fsSL https://raw.githubusercontent.com/weby-homelab/antigravity-cli-statusline/main/install.sh | bash
2

Verify settings.json

Перевірка налаштувань settings.json

Ensure that the statusline is activated in your configuration file ~/.gemini/antigravity-cli/settings.json:

Переконайтесь, що у вашому конфігураційному файлі ~/.gemini/antigravity-cli/settings.json активовано статуслайн:

{ "statusLine": { "type": "", "command": "/home/user/.antigravity/statusline.sh", "enabled": true } }
1

Run the installation PowerShell script

Запуск інсталяційного PowerShell скрипта

Launch PowerShell as your current user and execute:

Запустіть PowerShell від імені вашого поточного користувача та виконайте команду:

powershell -NoProfile -ExecutionPolicy Bypass -Command "iex (irm https://raw.githubusercontent.com/weby-homelab/antigravity-cli-statusline/main/install.ps1)"
2

Windows settings.json setup

Налаштування settings.json у Windows

The installer configures settings automatically, but you can manually edit it at %USERPROFILE%\.gemini\antigravity-cli\settings.json:

Скрипт автоматично пропише шлях у конфігураційний файл, але ви завжди можете відредагувати його вручну за шляхом %USERPROFILE%\.gemini\antigravity-cli\settings.json:

{ "statusLine": { "type": "", "command": "powershell -NoProfile -ExecutionPolicy Bypass -File \\"C:/Users/user/.antigravity/statusline.ps1\\"", "enabled": true } }

Compatibility Mode without Nerd Fonts

Режим сумісності без Nerd Fonts

If you use a terminal emulator that doesn't support Nerd Font glyphs (like default Linux TTY, ChromeOS Terminal, or Emacs), you can activate Classic Mode. This mode replaces icons with standard ASCII/Unicode symbols.

Якщо ви використовуєте термінал, який не підтримує розширені гліфи Nerd Fonts (наприклад, стандартний TTY, ChromeOS Terminal або Emacs), ви можете увімкнути Classic Mode. Цей режим замінює складні іконки на стандартні ASCII-символи.

To enable it, append the --classic flag to the command inside your configuration file.

Щоб увімкнути режим сумісності, додайте параметр --classic до команди виклику в конфіг-файлі.

Example settings.json with --classic parameter
Приклад settings.json з параметром --classic
"statusLine": { "type": "", "command": "/home/user/.antigravity/statusline.sh --classic", "enabled": true }
